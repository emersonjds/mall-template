
# Viva Connect

This project aims to improve the user experience when using cashback alongside a blockchain layer for a future currency.


## Stack utilizada

**Front-end:** React, Vite, TailwindCSS



## Instalação

Instale my-project com bun

```bash
  bun install 
  cd mall-template
  bun run dev
```
